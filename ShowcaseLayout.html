Class: ShowcaseLayout

ShowcaseLayout()

ShowcaseLayout is the main component in the Infographic UI app. All the other components are migrated from the storyboard ui app. The component uses 'react-grid-layout' npm libary for drag, drop & resize functionality are enabled using the same library.

generateDOM()

This function generates the DOM -
It generates the DOM according the "i" property-
  • interactivity_k_ - Toggle children interactivity
  • interactivity_x_ - Dynamic x/y axis
  • interactivity_f_ - Filter by lod interactivity
  • main_chart_ - Chart properties
  • legends_ - Legend container The function filters the different types of layouts (depending upon "i" of the layout) and then pushes the JSX element in an array.
    At the end, the function returns the entire array to the render method

    getLayoutState()

    Function to generate layout while rendering the app initially.
    Adds new layouts to existing array if new chart is added and delete if any previously existing chart is deleted

    Explanation of various conditions used in the function -

  • (result.data['react_resize_' + j].hasOwnProperty('children') && JSON.stringify(result.data['react_resize_' + j]['children']) - Checks whether toggle children are present with this chart or not.
  • (result.data['react_resize_' + j].hasOwnProperty('legends') && result.data['react_resize_' + j].legends === true) - Whether we need to display with this chart or not
  • ((result.data['react_resize_' + j].hasOwnProperty('x_axis_values') || result.data['react_resize_' + j].hasOwnProperty('y_axis_values')) - whether any column is dynamic or not
  • (result.data['react_resize_' + j].hasOwnProperty('filtercolumn')) - whether chart has filter by lod interactivity
    If any of the above conditions is true, then we will need to render another div for each element Following conditions are checked-
  • Whether layout data is present, if data is not present then It will create default layout array
  • If layout data is present and if any new chart/interactivity/legend is added, then it will add new default value for that element.
  • If layout data is present and if any existing charts/interactivities/legend is deleted, it will delete the layout corresponding to that element
