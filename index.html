App Setup

The application is built on node version 12.16.1 using React version 16.13.1. To run the application, one needs to have node on his/her computer.

Running the application for the first time

While running the application for the first time, one will need to install various dependencies. This can be done by following steps:

  • conda create -n dss-dev python=3.6
  • conda activate dss-dev (or) activate dss-dev
  • pip install -r requirements.txt --trusted-host=dss-new.pfizer.com:10000
  • conda install -c conda-forge nodejs=15.10.0
  • npm install

Running the application after first time

  • conda activate dss-dev && app.bat (or) activate dss-dev && app.bat
